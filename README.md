# Digitales Ökosystem Management

## Projektübersicht
Dieses Projekt simuliert ein **digitales Ökosystem** mit verschiedenen Objekten, Tieren und Lebewesen.  
Jedes Element hat **Parameter, Funktionen und Aufgaben**, um Interaktionen und Dynamik im Ökosystem zu ermöglichen. Das wurde zur Übung erstellt und auch zur weiterentwicklung für OOP

## Funktionen
- Verwaltung von Lebewesen, Tieren und Objekten  
- Individuelle Parameter 
- Aufgaben und Funktionen für jedes Element  
- Interaktionen zwischen Lebewesen und Umgebung
